General Information

Support for MongoDB database.

MongoDB is an open-source document-oriented NoSQL database. BSON is a binary serialization JSON-like format used to store documents and make remote procedure calls in MongoDB. This toolkit provides MongoDB insert/query/update/delete functionality where SPL to/from BSON conversion is generated automatically.

Below a table for BSON/SPL conversion: 

BSON	SPL

BinData	blob
Bool	boolean
N/A	complex
Date	timestamp
Double	decimal/float64
Int	(u)int8
Int	(u)int16
Int	(u)int32
Long	(u)int64
String	enum/rstring/ustring/xml
Array	List/Set
Object	Tuple/Map
Third-party libraries
  • This toolkit embeds mongodb headers/libraries under impl/<include|lib>.
